# mmgoget

A 'go get' alternative to work with module paths and project versions differ at major versions.

## Install

`go install github.com/isacikgoz/mmgoget`

## Usage

```sh
mmgoget github.com/mattermost/mattermost-server/v6@v7.2.0

```
